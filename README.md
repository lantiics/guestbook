# guestbook

This is the guestbook I created (& use) for my website https://lantics.me.

To use it, require the `guestbook.js` file in your expressjs server. The files in the `public` directory should work seamlessly, and can be used as a good starting point if you intend to customize anything.
You can also repeat the above instructions with `admin.js` and the `admin` directory.
